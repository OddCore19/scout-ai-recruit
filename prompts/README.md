# Prompt Catalog

This directory stores reusable prompt texts for the Scout AI Recruit flow.

Files:

- `jd_extraction_prompt.md`: current job description extraction prompt
- `candidate_extraction_prompt.md`: recommended candidate extraction prompt
- `candidate_extraction_prompt_legacy.md`: original candidate extraction prompt from the generated flow
- `fit_scoring_prompt.md`: current fit scoring prompt
- `recommendation_summary_prompt.md`: current recommendation summary prompt
- `structured_output_format_instructions.md`: current structured output formatting instruction

Recommended next use:

- replace the current candidate extraction prompt in the flow with `candidate_extraction_prompt.md`
- keep `candidate_extraction_prompt_legacy.md` only for comparison and rollback
